# 要件定義

## 1. プロジェクトの目的

プロ厨房HIT沖縄を「厨房機器販売会社」ではなく、店舗探し・開業準備・厨房設計・施工・設備導入・開業後・売却まで支える「飲食店づくりのパートナー」として認知してもらい、相談・見積依頼につなげる。

## 2. ブランドメッセージ

- キャッチコピー: 沖縄を飲食店から元気にする！
- 提供価値: 飲食店づくりを一社でワンストップ支援
- 信頼の根拠: 代表自身の開業・経営・設備投資・店舗売却の経験
- 独自性: ラショナル製品の現役ユーザーとしての提案力、全国ネットワークによる中古厨房機器調達、居抜き売買「ウリテン」との連携
- トーン: 信頼感、清潔感、プロフェッショナル。沖縄らしさは色やモチーフのアクセントに留める

## 3. 想定ユーザー

- 沖縄県内で飲食店を新規開業したい人
- 厨房機器の新規導入・入替え・買取を検討する店舗
- 厨房レイアウトや内外装工事をまとめて相談したい事業者
- ラショナル製品の導入効果を実務視点で知りたい人
- 居抜き店舗を探したい、または売却したい人

## 4. 公開サイトの必須機能

### ホーム

- 3枚構成のHEROスライダー
- 初期仮画像はグレー、アイボリー、ネイビーの3色で代替可能
- 自動再生、前後操作、現在位置表示、一時停止への配慮
- 主要サービス、強み、施工事例、最新情報、問い合わせへの導線

### サービス

- 新品・中古厨房機器販売
- 厨房機器買取
- 飲食店開業サポート
- 厨房レイアウト・CAD図面
- 搬入・設置
- 内装・外装工事
- ラショナル製品導入支援
- 居抜き売買「ウリテン」への外部導線
- ウリテン事業（居抜き物件紹介、店舗売却、店舗承継）
- 沖縄で飲食店を始める人向けの開業サポート（物件探し、地域事情、県外からの出店を含む）

### 施工事例

- 一覧、詳細、カテゴリ・エリアによる絞り込み
- メイン画像を含む最大10枚の画像、概要、対応内容、所在地表示
- 緯度・経度を使った沖縄マップ上のマーカー表示
- エリア区分は「北部・中部・南部・宮古・八重山・久米島／その他離島」の6区分
- 住所を公開しない案件では市町村レベルの表示と概略座標を利用可能

### 最新情報

- 一覧、詳細
- カテゴリ、公開日時、アイキャッチ、本文
- 下書き、予約公開、公開終了

### 問い合わせ

- 相談種別、氏名、会社・店舗名、電話、メール、相談内容
- 同意確認、入力検証、送信完了画面、管理者通知
- 迷惑送信対策

## 5. 管理機能の必須要件

- 認証された管理者のみ利用可能
- HEROの並び替え、表示／非表示、画像差替え、文言編集
- HEROごとのオーバーレイ色・濃度、ドット柄の表示・色・濃度・サイズ調整
- 施工事例の作成・編集・削除・公開管理、画像並び替え、緯度・経度入力
- 地図上のクリックによる座標入力とマーカーのプレビュー
- 最新情報の作成・編集・削除、公開予約
- 画像アップロード時の形式・容量検証と代替テキスト入力
- 一覧で公開状態と更新日時を確認可能

## 6. 非機能要件

- スマートフォンを優先したレスポンシブ対応
- キーボード操作、十分な色コントラスト、画像代替テキスト
- HEROは `prefers-reduced-motion` 利用者に自動再生を強制しない
- 適切な見出し構造、ページ固有title・description、OGP、構造化データ
- WebP/AVIF等の画像最適化、遅延読み込み、レイアウトシフト抑制
- CSRF、XSS、SQLインジェクション、ファイルアップロード対策
- 定期バックアップと復元手順

## 7. 未確定情報

実装中は仮値を許容するが、公開前に以下を確定する。

- 正式な会社表記、郵便番号、住所、電話番号、営業時間、定休日
- ロゴ・ブランドカラー・本部ブランドガイド
- 問い合わせ送信先と個人情報保護方針
- SNSアカウント
- 施工事例の実データ・写真・公開可能な位置精度
- 使用する地図サービスとAPIキー／利用条件
