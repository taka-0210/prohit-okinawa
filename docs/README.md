## 設計上の中心

サイトの主語は「厨房機器」ではなく「飲食店づくり」。開業前から設備更新、店舗売却まで伴走できることと、飲食店経営経験に裏打ちされた提案力を伝える。

運用面では、担当者が開発者に頼らず次の3領域を更新できることを最優先とする。

1. ホームのHEROスライド（画像・文言・オーバーレイ・ドット柄）
2. 施工事例（画像・本文・沖縄マップ座標）
3. 最新情報（公開日時・本文・公開状態）
